# Holberton School Web Front End

This repository contains static front-end exercises built around the **Techium** website project. The work is split by topic so each directory documents a different layer of the same site, from semantic HTML structure to the styled CSS version.

## Repository layout

### `html_advanced/`

Semantic HTML exercises for the Techium site. This directory includes:

- a finished homepage in `index.html`
- 37 numbered checkpoint files that show the page being built step by step
- supporting pages such as `about.html`, `contact.html`, `latest_news.html`, `article.html`, and `styleguide.html`
- favicon assets used by the HTML pages

### `CSS_advanced/`

CSS exercises for the same Techium site. This directory includes:

- a styled homepage in `index.html`
- 32 progressive stylesheet files in `styles/`
- image and favicon assets in `images/`

## Project context

The site content is organized around common agency-style sections:

- header and navigation
- hero banner
- services
- works / portfolio
- about
- latest news
- testimonials
- contact
- footer

## Usage

This repository does not require a build step. To review the work, open the HTML files directly in a browser, starting with:

- `html_advanced/index.html`
- `CSS_advanced/index.html`
