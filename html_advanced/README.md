# HTML Advanced

This directory contains the semantic HTML build for the **Techium** website. The files show the project being assembled incrementally, ending with a complete multi-section homepage and a small set of supporting pages.

## Contents

- `index.html`: final homepage markup
- `0-index.html` to `36-index.html`: numbered checkpoints for the homepage build
- `11-styleguide.html`, `13-styleguide.html`, `26-styleguide.html`, `28-styleguide.html`, `30-styleguide.html`, `32-styleguide.html`, `33-styleguide.html`, `34-styleguide.html`, `38-styleguide.html`, `39-styleguide.html`: step-by-step styleguide exercises
- `styleguide.html`: standalone HTML styleguide page
- `about.html`, `contact.html`, `latest_news.html`, `article.html`: supporting page templates
- `favicon.ico`, `favicon.png`: favicon assets

## Final homepage structure

The completed `index.html` is organized with semantic sections for:

- header and navigation
- hero
- services
- works
- about us
- latest news
- testimonials
- contact
- footer

## Purpose

The files in this folder focus on HTML structure and content organization:

- semantic page layout with `header`, `main`, `section`, `article`, `nav`, `aside`, and `footer`
- accessible document metadata such as title, description, language, and viewport configuration
- content components such as lists, tables, blockquotes, media, and links

## Usage

Open any file directly in a browser. For the finished version, start with `index.html`. To review the progression of the exercise, compare the numbered files and the styleguide checkpoints.
