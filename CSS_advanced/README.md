# CSS Advanced

This directory contains the styled version of the **Techium** homepage. It builds on the HTML project by adding typography, layout, color, reusable component classes, and image assets for the finished landing page.

## Contents

- `index.html`: final styled homepage
- `styles/1-style.css` to `styles/32-style.css`: incremental stylesheet checkpoints
- `images/`: portfolio, article, team, about, logo favicon, and other visual assets used by the page

## Styling context

The final page loads:

- Google Fonts for **Open Sans** and **Raleway**
- `styles/32-style.css` as the latest stylesheet
- image assets from `images/`

The CSS in this directory includes:

- a Normalize.css import
- custom properties for colors, typography, spacing, and transitions
- base element styling for text, headings, links, and buttons
- layout helpers such as containers, rows, columns, and section wrappers
- component styling for navigation, service cards, work cards, blog cards, and testimonials
- light and dark section theming

## Page sections

The styled homepage covers the same content areas as the HTML version:

- hero
- services
- works
- about
- latest news
- testimonials
- contact

## Usage

Open `index.html` in a browser to review the latest styled version. To follow the CSS progression, compare the numbered files in `styles/`.
